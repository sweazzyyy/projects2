# Инструмент визуализации графа зависимостей Python-пакетов

## Общее описание

Данный инструмент предназначен для анализа и визуализации графа зависимостей пакетов Python (pip). Программа позволяет:
- Анализировать зависимости указанного пакета
- Строить граф зависимостей с учетом транзитивности
- Визуализировать результаты в формате PNG
- Работать как с реальными репозиториями, так и с тестовыми данными

## Функции и настройки

### Конфигурация
Настройки задаются через TOML-файл со следующими параметрами:
- `package_name` - имя анализируемого пакета
- `repo_url` - URL репозитория или путь к тестовому файлу
- `test_mode` - режим работы с тестовым репозиторием (true/false)
- `output_file` - имя генерируемого файла с изображением графа

### Основные возможности
1. **Сбор данных**: Извлечение информации о прямых зависимостях пакета
2. **Построение графа**: Алгоритм DFS без рекурсии
3. **Обработка циклов**: Корректная обработка циклических зависимостей
4. **Тестовый режим**: Работа с файловыми описаниями графов
5. **Визуализация**: Генерация PNG через язык диаграмм D2

## Сборка и запуск

### Требования
- Python 3.8+
- Установленные зависимости: `pip install -r requirements.txt`
- Утилита D2 для визуализации

### Команды
```bash
# Клонирование репозитория
git clone <repository-url>
cd <project-directory>

# Установка зависимостей
pip install -r requirements.txt

# Запуск приложения
python main.py --config config.toml

# Запуск тестов
python -m pytest tests/
